# Project Office Globalfleet

Vue 3 + Vite

## Installation

Install Node.js and Install Yarn.

Package install:

```sh 

yarn install

```

Run development:

```sh 

yarn run dev --port 3010

```

Compile to production use:

```sh 

yarn build

```
